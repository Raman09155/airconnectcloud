What Would You Like To Know?

What is VPS hosting?

A Virtual Private Server (VPS) is one of the most adaptable web hosting options. You'll get a private partition on a physical server with committed resources. You can design, control, and customize your virtual server as per your inclination. Therefore, it is a great choice for experts looking forward to power and flexibility for their projects.
Notwithstanding the VPS being independent, and you being responsible for each work within the server, our team deals with the back-end business, directs the physical servers and guarantees uptime, speed, and dependability

How many sites can I host?

In CloudTreno Cloud Server, you can host unlimited sites until you fully consumed your resources. A moderate dedicated server configuration can run approximately 150 to 250 websites. You can seamlessly upgrad to increase number of site.

Can I get full control of the server?

Yes, you can get a full control of the server. You will have root access of your server so that you can install custom applications as per your needs. CloudTreno Linux VPS Server can be managed remotely via SSH and Windows Server via RDP.

Can I install my software on the VPS server?

Yes. Since a VPS works like a dedicated server, you can install any software products you need.

What is the difference between shared hosting and VPS or dedicated hosting?

Shared hosting is suitable for small projects, similar to portfolios or personal blogs. On account of Shared hosting, you share the resources and server space with different users. It implies that the server settings are not available to you and your site's performance can be affected by other users' activities.

Can I install cPanel or Plesk?

Yes, We provide cPanel with our Linux Cloud hosting Server Plan and Plesk with our Windows Server Plan. We also provide cPanel/Plesk Support. Please contact our support to setup cPanel/Plesk on your server.
